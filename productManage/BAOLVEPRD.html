      # 工业总结

      # Q1：工业项目能为用户提供什么？

      • 电子企业档案：结合地理信息，形成电子企业档案

      • 基础统计：提供强大的自定义统计功能报表

      • 企业洞察：通过模型以及大数据对企业，进行多角度分析洞悉企业发展状况

      • 企业监控：帮助用户快速定位问题企业

      • 企业标签：分析企业特性，为企业贴标签

      • 专题榜单：根据提前定制规则定期向用户推送榜单

      # Q2：从工业项目我&我们学到什么

      • 环比、同比：
        • 环比：环比是某个阶段与其上一个时长相等的阶段做比较。对于高速增长型业务，能非常好的体现出业务的增长趋势和事件的影响。
        • 同比：同比是今年的某个阶段与去年的相同时段比较。可以去除大多数业务的季节因素。
      • POI分层模式
      • 多级结构在地图上的表现形式
      • 设计时明确系统主线
      • 常用图表类型：柱状图 折线图 雷达图 饼图 （已和UI洽谈形成相应的规范）
      • 常用功能：测量（线、面、经纬度） 图层 全屏 全局搜索
      • 统计报表所应该具备的常用功能：数据列展示、基础筛选查询、快速排序、导出、自定义字段、快速编辑、单表格宽度可拖动
